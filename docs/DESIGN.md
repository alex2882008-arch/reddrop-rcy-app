# Design

This project follows a minimal visual system:

- Focus first: the primary action is always obvious.
- Quiet surfaces: avoid heavy gradients and visual clutter in core workflows.
- Strong readability: clear type scale and contrast for dashboard-heavy usage.
- Consistent rhythm: spacing and component density stay predictable.
- Status clarity: success/warning/error states are explicit and immediate.

## UI direction

- Neutral base with restrained accent colors
- Lightweight motion only where it supports user understanding
- Functional component patterns over decorative complexity

## Accessibility baseline

- Keyboard-reachable controls
- Sufficient contrast for text and action buttons
- Meaningful labels for form and dialog interactions
